## Website Performance Optimization portfolio project

**Check the performance of the site:**

- Clone the project on your desktop
```bash
git clone git@github.com:sterlingrobot/perfmatters.git
```
- Install the dependencies
```bash
npm install
```

- Run Grunt tasks
```bash
grunt
```
